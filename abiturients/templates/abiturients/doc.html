Документація

Даний проект було збудовано з метою надати абітурієнтам інформацію про їхнє перебування в рейтингу абітурієнтів а також показати інформацію про ці заклади та спеціальності в них.

Для побудови даного проекту було використано багато різних технологій. Зокрема для розробки лицьової частини сайту (фронтенду) було використано мову розмітки HTML, таблицю стилів CSS, мову програмування JavaScript з її бібліотеками Jquery (https://jquery.com/ )та Chart.js (https://www.chartjs.org/), також CSS фреймворк Bootstrap 4.0 який розроблений компанією Twitter для полегшення роботи з адаптивністю сайту. Детальнішу інформацію про фреймворк можна отримати за посиланням https://getbootstrap.com/.

Для реалізації серверної частини сайту(бекенд) було використано мову програмування пайтон, оскільки вона непогано себе показує в веб розробці і її веб-фреймворк Django 2.2.1. https://www.djangoproject.com/Щоб отримати детальнішу інфорацію перейдіть за посиланням https://www.python.org/ . З реляційних баз даних було використано PostgresSql 10.0 https://www.postgresql.org/.

Для зберігання даних в базі даних було побудовано 5 основних таблиць: abiturients_region, abiturients_specialty,abiturients_vnz,abiturients_subject, abiturients_abiturient_statement. До дотаткових таблиць належать таблиці створені зв’язками основних таблиць, а також таблиці створені фреймворком Django по замовчуванню auth_user, auth_group, permissions, sessions для підтримки аунтефікації,авторизації та сесій. Основні налаштування бази даних знаходяться на малюнку. {% load static %}

Abiturient Statement

Фреймворк Django працює за принципом MVT аналог MVC тільки замість терміна Controller в Django прийнято вживати термін Template. Так база даних з таблицями вже готова, потрібно ці дані якось представити. Для цього було побудовано спеціальні функції(Views).

Дані було використано з сайтів http://www.vstup.info/,https://abit-poisk.org.ua/.

Так як API у них відсутнє потрібно якось спарсити цю інформацію. Для цього була знову використана мова python з її бібліотекою BeautifulSoup 4.0. https://www.crummy.com/software/BeautifulSoup/bs4/doc/

Для візуалізації даних було використано бібліотеку Chart.js. В ній є багато готових шаблонів для діаграм і графіків.

Для побудови діаграми ми вибираємо 5 спеціальностей з найбільшою к-стю студентів та сортуємо їх по іменах потім візуалізуємо ці дані з типом Bar.

Для побудови графіка полігону частот ми вираємо всіх абітурієнтів з заданої спеціальності, сортуємо їх по середньому балу. Розбиваємо бали на проміжки і підраховуємо кількість студениів які входять в ці проміжки. Потім рахуємо середнє арифметичне кожного проміжку і передаємо дані через JSON формат і будуємо по цих даних графік з типом “line”.
